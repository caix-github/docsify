# 团建之百草园
2023-04-16
--- 
今天是有点燥热的天，阳光格外灿烂。与往常一样，完成洗漱等固定流程后提前了时间出门。毕竟是旅游放松的一天，心情丝毫不受燥热的影响，跟阳光一样灿烂。
到集合地点附近后，去附近吃了个早餐，可能是时间充裕，可以慢慢享受美食佳肴。

随着时间的推进，同事们也渐渐来到了集合地点。我们在车站那里唠嗑了一会，我们的领队点了个名，通知司机一声后，车也片刻就到。我们带着自己的东西开启了
一天的旅途。
