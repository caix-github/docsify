# 树叶
---

霜降了，阳光明媚，微风和煦，

追风的树叶跟树道别，徐徐落下。

枝头的树叶在阳光下更加青翠鲜活、

干枯的树叶被风卷掷发出清脆声响、

山头的树叶躲在山的一隅纹丝不动、

溪流的树叶在流水淙淙中远航前行。
