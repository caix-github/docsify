# 朝霞
---

那金黄的霞光

为矗立的山峰披上金纱

如同微醺的你

泛红的脸蛋

如同街上的恋人们

一切都是那么温柔绻缱

那金黄的霞光

给宁静的湖面撒下金点

如同爱笑的你

永远的璀璨

如同初次碰见了你

一切都是那么妙不可言
