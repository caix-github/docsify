# 枷锁
---

在城市繁华的幕后里，

大街小巷上车水马龙，

挤得有些密不透风。

每个人仿佛套着枷锁，

浑浑噩噩地奔波忙碌着。

上班疲惫性干活，

下班报复性疲惫，

日复一日，

心也不知桎梏何处！
