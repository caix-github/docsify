# 书
---

它搭起内心对话的桥梁，引领我们走进内心生活。拿起传记，走过作者的人生写照长廊，汲取生活经验。拿起散文，仿佛在知了的夏天，躺在吊床倾听蝉鸣的惬意。在琐碎的生活里注入点点滴滴藏匿的快乐。拿起历史，划过历史长河，感受泱泱华夏文明与外界船只的碰撞。拿起诗集，在诗意的国度里，云雾盛长，碧水盈盈。感受一唱三叹，余音绕梁，回味悠远。时间不紧不慢，将人熬成各种形状。即使被生活压弯了腰，也要博观而约取，厚积而薄发
