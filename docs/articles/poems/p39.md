# 落叶
---

泛黄的树叶

在风的吹拂中

自由自在地飘落

有各种各样的形状

宛如弧形的扁舟

宛如折皱的纸片

宛如卷卷的麻花

风儿再次光顾

它们便奏起乐章

沙沙作响

那是清脆的声音

贴近自然

恬进心间
