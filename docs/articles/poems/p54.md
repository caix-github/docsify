# 生命力
---

梅花自古苦寒来，

小草破土茁壮长，

那便是生命力最好的诠释。

那是一种快乐，

是春风得意马蹄疾，

一日看尽长安花。

那是一种自信，

是天生我材必有用，

千金散尽还复来。

拥抱阳光，

一路前行，

让生命力跃然纸上。
