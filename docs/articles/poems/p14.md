# 暖春
---

冬的脚步悄然离去

春的迎接便接踵而至

万物也纷至沓来

齐聚同庆此时此刻

瞧，不知绿叶谁裁出

原来是二月春风似剪刀

竹林外桃花三两支

春江水暖鸭先知

瞧，天街小雨润如酥

又有草色透看近却无

在春的主场里

雨也不曾缺席

初春的小草有雨珠的滋润

所以，在春的世界里，从不缺美

美得让人不禁浮想联翩
