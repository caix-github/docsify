# 两小鸭
---

桌前的两小鸭

颜色不相径庭

一只有翠绿的长帽

一只有兔耳的发箍

一只向北盼

盼似黄河之水天上来

一只向南望

望似不尽长江滚滚来

一只颜色金黄

看似活泼开朗

一只颜色暗紫

看似沉稳冷静
