# 自知

生活如饮水一般，冷暖自知。鲁迅先生在《而已集》里写过这样一段话：“楼下一个男人病得要死，那间壁的一家唱着留声机；对面是弄孩子。楼上有两人狂笑；还有打牌声。河中的船上有女人哭着她死去的母亲。人类的悲欢并不相通，我只觉得他们吵闹。”成人的世界里如弹簧绷着，生怕一断分崩离析。

曾几何时，我们也有梦，关于文学，关于爱情，关于穿越世界的旅行。而如今我们深夜饮酒，觥筹交错，杯子碰在一起，都是梦破碎的声音。
