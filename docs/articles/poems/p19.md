# 假如没有眼泪
---

假如没有眼泪

情感世界便失去了一样色彩

犹如画龙无法点睛

传说中的神龙便无法问世

假如没有眼泪

情感世界便失去了一种流露

心情无法溢于言表

那心灵世界便少了一扇窗

假如没有眼泪

情感世界便失去了一种真情

难以感化刃如秋霜之剑

最终停滞于为爱前行之路
