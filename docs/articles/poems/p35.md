# 星空
---

一闪一闪，亮晶晶，

满天都是小星星。

璀璨星空，

总有一群星星格外闪亮，

大部分星星日常的闪亮。

而部分有些黯淡的星星，

虽然被囿于一隅，

却用足自己的劲发着光，

共同编织这一张灿烂的星网，悬挂高空。
