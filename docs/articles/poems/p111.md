# 立秋
---

始为立，

熟为秋。

有着成熟的颜色，

那零星金黄，

轻染上繁荫浓绿。

点缀了千里绿野。

透露着秋的韵味，

凉风至、

白露生、

寒蝉鸣。

秋，又携带着收。

中稻开花结实

大豆结荚，

玉米抽雄吐丝，

棉花结铃，

事务倥偬之际，

请适时小憩，

让丝丝秋风，

为你助力！
