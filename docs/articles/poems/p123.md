# 老饕
---

观其色，精致形，赏心悦目。

嘎嘣脆，如甘霖，滋润心地。

滋滋响，动其声，垂涎欲滴。

味其绝，牵其魂，蠢蠢欲动。

入其口，开味蕾，心花怒放。
