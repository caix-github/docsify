# 与朋友舒适相处的背后，都是不动声色的接纳
---

接纳朋友的善意，不担忧互相亏欠。因为彼此的交情都是从互相亏欠的缝隙里升温的。
太过泾渭分明，反而会疏远人心。
接纳朋友的不同，每个人都独一无二的，有自己的性格和风格。海纳百川，有容乃大。
既能加固友谊，也能拓宽自己的胸襟。
接纳朋友的优秀，不吝啬欣赏。不仅认同了他，也是关系最好的滋养
